# Portfolio Project

**Author:** [Yash Sharma]

## Description
This project is a personal portfolio website designed to showcase my work, skills, and projects. The approach focuses on a clean, modern design using HTML, CSS, and JavaScript for interactivity and responsiveness.

## Features Implemented
- Responsive layout for desktop and mobile
- Project showcase page
- Custom background image
- Interactive navigation
- Smooth scrolling and transitions

## Steps to Run the Project
1. Clone the repository from GitHub:
   ```
   git clone <your-repo-url>
   ```
2. Navigate to the project folder:
   ```
   cd Portfolio
   ```
3. Open `index.html` in your web browser to view the portfolio.
